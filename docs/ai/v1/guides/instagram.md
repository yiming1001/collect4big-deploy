# Instagram

Instagram 内容与话题采集

- 平台 ID: `instagram`
- 分类: 海外社媒与流媒体
- 能力数量: 4

## 能力

- `instagram_post_search`: 综合搜索。按关键词搜索 Instagram 内容
- `instagram_user_posts`: 用户帖子。采集 Instagram 用户发布的帖子
- `instagram_hashtag_search`: 话题标签。搜索 Instagram 话题标签
- `instagram_post_detail`: 帖子详情。按帖子 ID 获取 Instagram 内容详情

具体输入参数、分页、字段和去重规则以同平台 JSON 能力包为准。
